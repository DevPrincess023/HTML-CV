# Personal HTML-CV
A fully structured personal CV created using semantic HTML. Demonstrates knowledge of HTML elements, layout organisation, and clean code practices.

## Features
-Simple and clean structure
-Easy to read
Fully coded from scratch

## Learning Points
-Structuring content with HTML tags
-Oganizing a CV layout without CSS
-Making semantic HTML
